•	Developed a role-based task management platform enabling administrators to assign tasks to different user roles and track progress in real-time.
•	Implemented secure authentication and authorization using JWT-based login and role-based access control (RBAC).
•	Built modules for task creation, assignment, status tracking, and record management.
•	Designed RESTful APIs using Express.js and integrated MongoDB for efficient data storage.
•	Developed a fully responsive frontend using React.js for seamless task monitoring and management.
•	Maintained historical records of task assignments and completion status for reporting and accountability.

# Role Based Task Management System

## Tech Stack
- React (Vite)
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Features
- Role-based authentication
- Task assignment
- Student management
- Admin dashboard

## Installation

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm run dev
